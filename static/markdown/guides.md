# General Name Rule

## variable

一般 variable - 小駝峰

```lang=javascript
var brianBoobSize = "big";
```

常數 const - 大寫加底線

```lang=javascript
const BRIAN_BOOB_NUM = 2;
```

## function

一般 function - 小駝峰

```lang=javascript
function isBrainBigBoob(){
  //...
};
```

## table name

大駝峰

> BrianBigBoob

## columns name

小寫加底線

> brian_big_boob

## spacing

> php: 4
> javascript: 2
